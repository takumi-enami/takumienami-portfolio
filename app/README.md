# Takumi Enami Portfolio

Next.js App Router based static portfolio site for Cloudflare Pages.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run start`

## Deployment

This project is configured with `output: 'export'`, so `npm run build` generates a static site in `out/`.
Deploy `out/` to Cloudflare Pages as a static project.
