PRODUCT IMAGE SIZE GUIDELINES

[1] Product card thumbnails
- Aspect ratio: 4:3
- Recommended size: 1600 x 1200 px
- Minimum size: 1200 x 900 px
- Usage: PRODUCTS section card thumbnails
- Notes: Keep all card thumbnails in the same ratio for visual consistency.

[2] Product detail hero image
- Aspect ratio: 4:3
- Recommended size: 1600 x 1200 px
- Alternative: 1800 x 1350 px
- Usage: Main visual at the top of each product detail page
- Notes: This should represent the product at a glance.

[3] Product detail gallery images by category

Web app / desktop UI
- Recommended aspect ratio: 8:5
- Recommended size: 1600 x 1000 px
- Alternative: 1600 x 1200 px
- Usage: Dashboard, management screen, workflow screen

Mobile app screen
- Recommended aspect ratio: 9:16
- Recommended size: 900 x 1600 px
- Alternative: 1080 x 1920 px
- Usage: Smartphone-only UI, mobile-first flows

Cross-platform app
- Prepare both of the following when possible:
  - Desktop/web view: 1600 x 1000 px
  - Mobile view: 900 x 1600 px
- Usage: Products that have both browser and phone experiences

Hardware / device photo
- Recommended aspect ratio: 4:3
- Recommended size: 1600 x 1200 px
- Alternative: 1800 x 1350 px
- Usage: Physical products, equipment, prototype photos

3D print / object photo
- Recommended aspect ratio: 4:3
- Recommended size: 1600 x 1200 px
- Alternative: 1:1 at 1400 x 1400 px
- Usage: Printed parts, enclosures, models

Diagram / flow / structure image
- Recommended aspect ratio: 4:3 or 3:2
- Recommended size: 1600 x 1200 px
- Alternative: 1800 x 1200 px
- Usage: Architecture diagrams, process flow, system overview

[4] Recommended future image model
- Product card thumbnails should remain fixed at 4:3.
- Product detail images should later support multiple aspect types:
  - landscape
  - portrait
  - square

[5] Current quick rule
- Card thumbnail: 1600 x 1200 px
- Web/desktop detail image: 1600 x 1000 px
- Mobile detail image: 900 x 1600 px
- Hardware/3D detail image: 1600 x 1200 px
